Welcome to OntoMathPro Ontology Page!

The OntoMathPro ontology has been developed by the research group from Kazan Federal University. The ontology is geared to be the hub for math knowledge in the Web of Data. We shared the sources with the Semantic Web community to engage our colleagues from elsewhere in its further development. We are going to create an ecosystem of datasets and mashups around the ontology.

How to Contribute?

  1. Clone the develop branch.

  2. Edit OntoMathPro.omn using your favorite editor (e.g. Protege) and save it in OWL 2 Manchester Syntax.

  3. Create a pull request for the develop branch and send it.
